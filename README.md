# Android CI Template

GitHub Actions Cloud Build for Android

## Features
- Auto build Debug/Release APK
- Java 17
- Gradle caching
- Auto artifact upload

## Usage

1. Push code to this repo
2. Go to Actions tab
3. Download APK from artifacts

## Directory
```
.github/workflows/android.yml
```

## Note
- Free: 2000 mins/month
- Debug APK: 7 days retention
